    • Open Source Designers Workstation

     

    Description

    The aim of this project is to make an appliance with suse studio that caters to the majority needs of visual creative designers. it will look fantastic, contain only what is needed for most design tasks and be ready to use instantly for a wide variety of design workflows. The project requires the following

    • Building some packages (marked with * )
    • Setting up overlay files
    • Documentation and walkthroughs
    • Testing
    • Website + artwork
