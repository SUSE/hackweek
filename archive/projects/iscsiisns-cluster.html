    • iSCSI/iSNS Cluster

    Tags: Idea, storage

    Description

    Using a low cost Serial Attached SCSI (SAS) shared storage disk array, create an HA-Cluster providing iSCSI Logical Units e.g. for serving storage to individual virtual machines. Additionally, demonstrate how Heartbeat2's iptables based cluster IP alias can be used to load balance iSCSI sessions across the cluster.

    People

    Robert Wipfel originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
