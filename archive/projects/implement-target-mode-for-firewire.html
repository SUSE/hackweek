    • Additional Target mode backends

    Tags: InProgress, storage, kernel

    Description

    With 10.3 we've added the tgt generic target framework to OpenSUSE. This currently provides support for iSCSI and ibmvscsi. As this is quite limited it would be great to enhance this with other backends.

    * FC Target Mode: There already is a FC Target mode driver for the QLogic HBAs, originally done for the scst project. So it should be doable to port them to the tgt framework. * Firewire Target: There is a stand-alone tool from Oracle, called 'endpoint'. This should be ported to the tgt framework * aic79xx: The current aic79xx driver has support for Target mode, although this is currently disabled. Would be fun to enable it and hook it in the tgt framework.

    People

    Hannes Reinecke originated this idea and will be working on it. Help will be very much appreciated. Sebastian Vollath will test linux fc targets.
