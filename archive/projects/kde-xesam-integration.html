    • KDE Xesam Integration

    Tags: InProgress, KDE, Xesam

    Description

    Xesam is an umbrella project with the purpose of providing unified apis and specs for desktop search- and metadata services. One of the goals of the xesam project is to provide a common dbus API for desktop search engines.

    I want to explore Xesam and what engines support it already, test/package xesam-tools and hack some KDE integration for it (eg. Kickoff-Plugin, KIO slave or full featured search-frontend).

    People

    Stephan Binner originated this idea and is working on it.
