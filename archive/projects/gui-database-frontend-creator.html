    • GUI database frontend creator

    Tags: Idea, desktop

    Description

    A tool to create (read: click together) simple databases and create beautiful custom GUI frontends for them. Target group: home users who want a simple database for recipes, books or the addresses of their local Kaninchenzuechterverein. Bonus: a standalone player for these databases to give them away, export of parts of the data for letter printing to OpenOffice. Or create a database with multimedia content, create a standalone player, a live CD, and have a complete kiosk information system with a few clicks!

    For the backend probably one of the existing opensource databases could be used.

    People

    Sonja Krause-Harder originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
