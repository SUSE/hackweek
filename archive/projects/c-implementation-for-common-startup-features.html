    • C Implementation for common startup features.

    Tags: Idea, Performance, SLES, SLED, openSUSE

    Description

    Many of the shell scripts used during boot time spend a lot of time launching hundreds of helper programs to start a service or shutdown a service.

    We should rewrite some of the basic services and write C implementations for some of those services to speed up boot time.

    This would allow the parsing and reparsing of a lot of configuration files, hundresd of extra commands from being executed.

    People

    Miguel de Icaza originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
