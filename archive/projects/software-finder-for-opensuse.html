    • Software Portal for openSUSE

    Tags: Community, openSUSE, InProgress, judgecommunity, judgeme

    Description

    Write a tool which makes it easy and fun for the end user to find software for openSUSE.

    This could be a client to the build service which browses software which fits to the installed system. It could try to collect additional data from software index sites like Freshmeat or the *-apps.org sites or embed a preview of the software homepage like some web sites do it these days when hovering links. The tool should integrate with software installation, so that the user can easily install found software.

    Another approach would be to extend the YaST package management module to also cover uninstalled software, show additional data like screenshots and allow the user to directly start applications from the packag management view, so that it's easy to try out new software.

    See also http://opensuse.org/Software_Portal

    People

    Cornelius Schumacher originated this idea. Pascal Bleser, Benjamin Weber, Justin Haygood and Francis Giannaros are working on this idea.
