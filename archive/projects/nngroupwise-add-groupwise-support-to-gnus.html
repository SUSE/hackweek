    • nngroupwise: Add GroupWise support to Gnus

    Tags: Idea, mail, emacs, groupwise

    Description

    A lot of people spend their hacking lives within Emacs, and several use Gnus as their primary mail reader. We currently have to use the IMAP and SMTP interfaces, both of which are discouraged by IS&T.

    It would be great if we could integrate GroupWise support into Gnus directly. Gnus already has, albeit quirky, support for non-local mail storage (//nnimap//), and we may be able to use the abstractions in there for GroupWise support.

    In terms of integration, I see the following tasks:

    • Add a ''groupwise-send-it'' function to send outgoing using GroupWise
    • Adapt the //nnimap// backend to use the GroupWise protocol for mail viewing
    • Adapt the //nndiary// backend to integrate with the GroupWise calendar

    People

    Raja R Harinath originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
