    • KDE TextCompletion History Editor

    Tags: Desktop, KDE, openSUSE, SLED, Done

    Description

    Within KDE the KHistoryCombo class offers text-completion while typing. This is used eg in the URL bar of Konqueror and web pages form fields. Currently the context menu of these widgets offers only to clear their whole history. If you mistyped an input (eg your name or email address, or an URL) there is no way to remove just the faulty value but it may keeps reappearing and annoying you. The idea is to implement a text-completion history editor.

    Stephan Binner originated this idea and worked on it. Packages with it were submitted to abuild.
