    • Multitouch display

    Tags: Idea, Desktop, Graphics, hardware

    Description

    Build hardware and software for a multitouch display demo, a touch screen that can be actuated with multiple fingers simultaneously, allowing for new user interface paradigms like:

    • Dragging the corners of a picture to shrink/enlarge it
    • Dragging the corners of a window to tilt it
    • Tossing windows and other objects with your hand
    • Sorting through a large number of media

    People

    Nat Friedman originated this idea. David Reveman is familiar with the X work to support multiple cursors; Nat Friedman is interested in this project
