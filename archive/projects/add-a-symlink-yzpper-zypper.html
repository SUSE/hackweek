    • Add a symlink yzpper -> zypper

    Tags: Idea

    Description

    SUSE systems alway had a symlink zast -> yast which comes in handy if you are using a german keyboard (QWERTZ) but have english keyboard layout set as default (or the other way round).

    I would find it pretty nice if we would have a similar symlink yzpper -> zypper for zypper.

    People

    Stefan Seyfried originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
