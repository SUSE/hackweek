    • Sound support for Hex-A-Hop (Game)

    Tags: Idea, games, openSUSE, Sound, Community

    Description

    Just found a great game with sources available. **Hex-A-Hop** is a puzzle game based on hexagonal tiles. Only lacking feature is sound IMHO. Would be cool to add some sound and getting familiar with SDL by doing this. Need to check the license of the current sources/content together with the author (Tom Beaumont <tombeaumont@yahoo.com>) first. Currently there is only a copyright notice available ("All content is Copyright 2005 Tom Beaumont").

    People

    Stefan Dirsch originated this idea. Pavol Rusnak is interested in implementing this idea.
