    • udev rules in YaST2 network

    Tags: InProgress, yast, network

    Description

    For each configured network interface (ethernet card, wlan card) there is hardware dialog in YaST lan module. Put here configurable widget from udev rule to rename interface name identified by MAC address. This name will be used also used as configuration name (ifcfg-eth0). It is already implemented in sysconfig.

    People

    Michal Zugec originated this idea and is currently working on implementing this idea. Christian Zoz already implemented sysconfig part.
