    • Robotic Tools for openSUSE

    Tags: Idea, openSUSE, DevTools, hardware, embedded

    Description

    I would like to have/work on a set of tools (esp. development tools) for the most common existing robot projects. For the beginning, focusing on the Lego Mindstorms set (old RCX and new NXT) would be more than enough.

    * Packaging of "Not Quite C" aka NQC for RCX * NXC for the the new hardware (NXT) * The Java firmware and the Java programming tools: LEJOS * packaging of BrickOS * easy set up of environement for ARM development * Mid term having an IDE/set like Microsofts Robotic Studio

    People

    Mihnea Istinie originated this idea. No one is currently working on implementing this idea (it's up for grabs!)... And I don't think that a lot of people are interested in robotics. However, I'll have the HW in the office during the week, so that everybody can play around.

    I've built a robot, which will be programmed to kidnap the best hackers and packagers for this idea. As you maybe already know, resistance is futile.
