    • Build a MugShot application to grab photos from a web cam

    Tags: Idea, Graphics, Video, Desktop

    Description

    This application should show a live preview from my webcam and take pictures and store them. It should be built so the widget that actually take the photos can be reused in the Desktop, in Banter, Evolutions, and F-Spot (or wherever else a mugshot is needed). It might be cool to have some effects on the photos like Apple's Photobooth.

    People

    Calvin Gaisford originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
