    • Fix up and package "tuxcursors"

    Tags: TryMe, InBuildService, Done, Graphics, Desktop, openSUSE, SLED

    Description

    {{ideas:watch-64x64-00.png|}} {{ideas:left_ptr-64x64-00.png|}} {{ideas:top_side-64x64-00.png|}} {{ideas:sb_h_double_arrow-64x64-00.png|}}

    There is a nice animated cursor set with little penguins available from http://www.kde-look.org/content/show.php/Resizeable+Tux+Cursor!?content=5376 Unfortunately it comes in binary only form, and the original author has lost the sources (PNG images and config files). Additionally, it has some flaws like the hourglass cursor having its hotspot on th e upper left edge instead of in the center of the hourglass.

    The idea is: "disassemble" the tuxcursors theme to PNG images, recreate config files and package it up in the openSUSE Buildservice, so that other artists can start improving the cursor theme.

    People

    Stefan Seyfried originated this idea.

    Related Materials

    The original theme is available from: http://www.kde-look.org/content/show.php/Resizeable+Tux+Cursor!?content=5376 A cursor editing program, it is very basic, but it can load binary cursors and extract the PNG images from them: http://freshmeat.net/projects/xcurs/
