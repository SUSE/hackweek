    • Media server Based on Directory as Backend in storing song/video identity

    Tags: InProgress, Server, Hacker

    Description

    making of a plugin based media server to serve (audio/video) to different client's based on DAAP or shoutcast or mythTV. here the identity of the media (song/video) is stored in the LDAP Directory. since a songs have a tree structure, the artist->(album1->song1/song2/song3)/(album2->song1/song2/song3), the same structure holds for videos too. hence they could be stored in a directory based tree. advantage of storing mostly read-only info in the directory makes fetching of data faster for playlist processing. The server design by making it plugin based, the plugins for DAAP/ shoutcast and mythTV can make this server to serve for different plugins.

    the songs can be accessed using banshee/itunes from the server. the video can be accessed using mythTv from server.
