    • Openstreetmap: Eval and document OSM related tools

    Description

    There are many tools around the http://wiki.openstreetmap.org/wiki/Main_Page project, from managing GPX files to creating customized maps.  Many tools are already packaged as part of the https://build.opensuse.org/project/show?project=Application%3AGeo, while other tools such as http://phpmygpx.tuxfamily.org/phpmygpx.php are difficult to package, and probably just need documentation.

    OSM book (in German): http://www.openstreetmap.info/de/news/2010-05-31-neue-auflage.html http://www.lob.de/isbn/9783865413758/
