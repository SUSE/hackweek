    • Full Amazon EC2 support

    Tags: InProgress, HW-Docs, Community, Virtualization, Web20, openSUSE

    Description

    Fully enabled openSUSE 11.0 to run on Amazon's Elastic Compute Cloud. This includes:

    • AKI/ARI for i386 & x86_64
    • Base AMI for both types
    • Developer AMI for both types
    • Enhancements to match Amazon same AMI experience
    • Adding other features from public AMIs
    • Full support for Elastic Block Service
    • Consider hosting repos on S3 to enable very fast and easy updates

    People

    Peter Bowen originated this idea. He is currently working on it and would love ideas and feedback. You can contact him via email or via irc (pzb on freenode).
