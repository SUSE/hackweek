    • Implement a moonlight/silverlight backend for the processing language

    Tags: Idea, Mono, Desktop, Graphics

    Description

    Processing is a very nice and simple language to produce images and animations. The current implementation compiles to a Java Applet. The idea is to implement a parser and an interpreter that can run over moonlight. It would provide access to a big repository of nice demos.

    The language is pretty simple and the interpreter would just generate the required animations. Even with that, you can see some very nice demos.

    People

    Rodrigo Kumpera originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
