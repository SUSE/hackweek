    • Add FastCGI interface to XSP

    Tags: Idea, Mono, Server, Web20, Network, HelpWanted

    Description

    Several web servers support the FastCGI interface for integrating external web languages, frameworks, etc. It would be really great if we could host ASP.NET applications on these servers. I started on it a while ago and had a generic FastCGI implementation in C# but have not hooked it up to XSP yet.
