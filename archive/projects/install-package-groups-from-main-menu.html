    • install package groups from main menu

    Tags: Idea

    Description

    http://lists.opensuse.org/opensuse-packaging/2008-02/msg00044.html

    Add an entry to each submenu to install packages that fit to the submenu (add icons there). E.g.

    Start > +--Games > +--Action > +--Arcade > +--Frozen Bubble +--Pingus ... +--Install more arcade games ...

    The "Install more ..." entry would launch the yast2 package manager with the RPM group "Amusements/Games/Action/Arcade" preselected.

    This would also be a motivation to clean up RPM groups in openSUSE packages.

    People

    Michal Marek originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
