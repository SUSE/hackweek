    • online source code collaboration tool

    Tags: Idea, collaboration

    Description

    As a student, every so often I'm working on some source code or helping someone else and wish that there was an easy way to collaborate 'live'. So my thought is either through some form of chat (maybe gaim) or web app to expose a page that has a source editor built in, for various languages, that would allow two to multiple users to code at once. Something like Google's docs/spreadsheet app.

    People

    Stephen Shaw originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
