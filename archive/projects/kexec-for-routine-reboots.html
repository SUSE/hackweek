    • kexec for routine reboots

    I want to have kexec support routine reboots.

    Tags: InBuildService, HW-Tester

    Description

    I added two variables (CMDLINE and KEXEC) to /etc/sysconfig/shutdown, that make /etc/init.d/halt use kexec when the system is rebooted. In comments in /etc/sysconfig/shutdown, there is an example about how to use that. You may also use the yast sysconfig editor to activate this feature.

    The package aaa_base that is needed for that should be available at [[http://software.opensuse.org/download/home:/azouhr/openSUSE_10.2/|http://software.opensuse.org/download/home:/azouhr/openSUSE_10.2/]] once the build is finished.
