    • Offline mode for KDE 4

    Tags: Desktop, Mobile, Done, kde

    Description

    I developed a set of offline mode tools for KDE 3 in a branch. Port this to KDE 4. This gives one place in [[http://solid.kde.org]|Solid]] for application authors to look for network status and adapt their apps' behaviour accordingly. Additionally provide test tools to simulate network disconnection without pulling cables out.

    People

    Will Stephenson originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
