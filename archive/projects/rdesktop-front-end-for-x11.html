    • rdesktop front-end for X11

    Tags: Idea, Desktop, Virtualization, Graphics, crossplatform, windows

    Description

    Expose our own X11 servers as rdesktop, very similar in spirit to Xvnc: basically allowing an Xserver to be presented to client machines as a remote Windows workstation.

    The underlying protocol for rdesktop has the advantage that it is bundled with every Windows machine and Windows sysadmins would be able to remotely connect to Linux hosts using the tools that are part of the operating system as well as doing seemless file sharing and audio redirection.

    People

    Miguel de Icaza originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
