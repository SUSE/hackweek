    • Google Applications "Packages"

    Tags: Idea, Desktop, openSUSE

    Description

    People

    Stephan Binner originated this idea. No one is currently working on implementing this idea (it's up for grabs!).

    Related Materials

    As far as I know Google Earth and Picasa are not allowed to be included in distributions. Mandriva ships with in the menu listed installation scripts (basically wget and starting the installer) to be run as user.

    My suggestion would be to create Build Service packages which run such script in %post, do run script magic to make it available for all users in the menus if necessary and also care about deinstallation in %postun.

    An idea worth trying to solve deinstallation is maybe be also to create ad-hoc an rpm of the applications with checkinstall and to install that rpm.

    PS: In the openSUSE survey 56.9% said that they want to have Google Earth included into openSUSE and 30.9% want Picasa.
