    • Disk Snapshots

    Tags: Idea, openSUSE, SLES, SLED, HelpWanted, Server, Desktop, Kernel, storage

    Description

    Using the device-mapper present in Linux 2.6, it's possible to create copy-on-write snapshots. Unfortunately, there is currently no way to merge that data back into the originating device. If we had that ability, we could do really cool stuff (filesystem transactions). See more in my blog entry [[http://www.snorp.net/log/2007/05/04/rpm-transaction-enhancements/|here]].

    People

    James Willcox originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
