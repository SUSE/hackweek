    • Ubiquitous transparent version control

    Tags: Idea, FileSystem, VersionControl, SLES, SLED

    Description

    Version control is an amazingly powerful concept which makes user sleep better, because there is no fear of accidently deleting data anymore. The drawback is that you have to explicitly put your data under version control which involves some setup work, choosing a version control system (which can be hard these days ;-)), and in many cases manually checking data in and out.

    An approach to solve this problem would be to make use of a versioning file system, so that all data on disk automatically is under version control. By adding support for it in the file dialogs many applications could easily benefit from that. Further integration with applications might still be desirable, though.

    People

    Cornelius Schumacher originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
