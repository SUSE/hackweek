    • finish ioquake3 VM for x86_64

    Tags: Done, games

    Description

    Quake3 mods are compiled from C code into the platform independent QVM format (quake virtual machine). When called the game compiles the QVM into native code using a built in 'JIT compiler'. ID software wrote compilers for i386 and ppc. Ludwig Nussel created one for ioquake3/x86_86 which uses gas as assembler. gas takes quite a while to compile the code. To really make x86_64 production ready ioquake3 needs to emit machine code itself.

    Additionally the QVM interpreter could be speed up to help platforms for which no built in compiler exists.

    People

    Ludwig Nussel originated this idea and is working on implementing this idea.
