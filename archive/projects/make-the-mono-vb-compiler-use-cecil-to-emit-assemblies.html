    • Make the Mono VB compiler use Cecil to emit assemblies

    Tags: Idea, DevTool, Mono

    Description

    Currently the Mono VB Compiler (vbnc) uses the Reflection.Emit API to emit assemblies, but the API has it's limitations, which would be overcome by switching to use Cecil.

    The single most important advantage would be that vbnc could emit programs targeting the v1.0 framework.

    People

    Rolf Bjarne Kvinge originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
