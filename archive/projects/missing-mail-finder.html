    • Missing Mail Finder

    Tags: Idea, Desktop, openSUSE, SLED, mail

    Description

    Have you ever been waiting for that urgent e-mail you were expecting, and it never comes? Or you go looking for the e-mail that should have been sent to you, and it isn't there? Because it is in the **spam** folder! :-(

    Idea: enhance the search function on one or more mail clients so that if you search on the Inbox and get few or zero hits, you then go search the spam folder.

    Simple version for Thunderbird: because spam filtering is done locally, the spam folder is easy to search.

    More complex version: if you can find an open-API for server-side spam filtering, then go query it.

    People

    Crispin Cowan originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
