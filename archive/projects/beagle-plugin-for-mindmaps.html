    • Beagle Plugin for mindmaps

    Tags: Idea, Desktop, Office, Graphics, vym, mindmap, openSUSE

    Description

    Beagle could search through mindmaps created by vym (https://sourceforge.net/projects/vym/). vym uses (zipped) XML to store the data, so it should not be too hard write a plugin.
