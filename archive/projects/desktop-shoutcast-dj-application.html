    • Castomatic - Desktop Shoutcast DJ Application

    Tags: InProgress, Desktop, Mono, Network, JudgeFailure

    Description

    Create a Mono based application that integrates with the Banshee player to shoutcast a playlist out for the world to consume. The application needs to mix voice input and support cross-fading of voice input and music. The end goal is a simple DJ system for Linux that could be published into Second Life. Boyd Timothy has volunteered to give a live performance on Friday if I can complete the project.
