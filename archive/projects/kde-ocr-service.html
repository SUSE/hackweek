    • KDE OCR Service

    Tags: KDE, graphics, OCR, desktop, InProgress, JudgeMe, JudgeDesktop

    Description

    Write a OCR (Optical Character Recognition) Service for KDE on base of the code Kooka uses for OCR.

    It calls an external, specialised OCR commandline tool (ocrad, tesseract), processes the result and does a dictionary lookup to lower mistakes. The results are displayed in a dialog that makes correction easy.

    The benefit of a general KDE service would be that every application that deals with images could use a standard ocr dialog to do text recognition on the image. That would probably be the first time ever something like that exists.

    End of Hackweek Status

    A KDE service for ocr that is usable from all KDE apps is available. It is still in prototype state but working and producing results. It can be used from all KDE apps dealing with images.

    It is not yet upstream, some issues have to be sorted out and fixed.

    People

    Klaas Freitag originated this idea. He is working on the implementation.
