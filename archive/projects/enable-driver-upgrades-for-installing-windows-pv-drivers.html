    • Enable driver upgrades for installing Windows PV drivers.

    Tags: InProgress, Virtualization

    Description

    Reduce reboots required during an update of the VMDP in Windows to 1 from 3. Currently to upgrade the PV drivers for Windows you must 1. uninstall the old drivers and then 2. Install the new version. This requires at least 3 reboots (4 on Windows 2000).

    People

    Lynn Bendixsen originated this idea. He and Jason Douglas are currently working on implementing this idea.
