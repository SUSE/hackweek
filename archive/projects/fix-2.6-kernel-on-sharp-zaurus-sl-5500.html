    • fix 2.6 kernel on Sharp Zaurus SL-5500

    Tags: Idea, Kernel, Mobile, embedded

    Description

    As Sharp Zaurus SL-5500 is nice PDA, but lacks Linux 2.6 support we want to add/fix the following features:

    • MMC card support
    • sound (alsa)
    • Sharp I/O port
    • charging
    • touchscreen

    People

    Cyril Hrubis originated this idea. Current workers:

    • Cyril Hrubis
    • Milan Vancura
    • Libor Pechacek
    • Erik Hamera

    Pavel Machek would be (hopefully) able to help.
