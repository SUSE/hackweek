    • Create web interface for YaST

    Tags: InProgress, Management, openSUSE, SLES, SLED, YaST

    Description

    YaST is a powerful tool, web-based interface would make the range it covers even better.

    I (joe) am hijacking this idea. There are several ideas around a YaST web frontend. What I am going to try is start from the outside in. That means I will write a web frontend with some sample management modules. I may or may not be able to finish hooking them up to actual YaST backend functionality.

    The aim of my project is to investigate technologies that can be used for the web frontend.

    I will use Zope3 with grok and maybe z3c.form. The basic concepts should be easy to port to other Python frameworks like Django or TurboGears as well as Ruby on Rails.

    People

    Jiri Srain originated this idea.

    Joachim Werner is working on one possible implementation of a YaST web frontend.
