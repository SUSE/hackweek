    • YCP to $[some popular virtual machine] compiler

    Tags: Idea, YaST, Management, Java

    Description

    YaST uses the YCP language. Bindings exists for writing modules in different languages. Making a YCP -> Java compiler (for example) would allow modules to be written in all languages that target that VM.

    People

    Duncan Mac-Vicar originated this idea and is doing some basic research on it. No one is currently working on implementing this idea (it's up for grabs!).
