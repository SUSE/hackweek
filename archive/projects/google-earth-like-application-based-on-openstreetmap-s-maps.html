    • Google Earth - like application based on OpenStreetMap's maps

    Tags: Idea, Desktop

    Description

    Combining OpenStreetMap's maps and Wikipedia entries, it should be easy to provide a Google Earth - like application, that shows the maps and, when requested, information about the places selected on the map, taken from Wikipedia.

    People

    Rodrigo Moya originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
