    • Simple software installation from Build Service

    Tags: Idea, openSUSE, BuildService

    Description

    Make it easy to get and install software from the openSUSE Build Service without fiddling around with repository URLs or other low-level stuff. There is some community work going on in this area. They follow the idea to use a web browser to find the software and then install it with a single click which invokes a YaST module which takes care of the rest. Might be a good base or source for inspiration.

    People

    Some people from the community are working on something like that (e.g. Pascal Bleser and Benjamin Weber).
