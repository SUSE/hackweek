    • Enable NIS over IPv6

    Tags: Idea, Network, Server, InProgress

    Description

    Using rpcbind and the TI-RPC library, implement NIS (aka YP) over IPv6. Initially, this means ypserv needs to be adapted to tirpc, and ypbind and ypwhich need to be adapted so that they support both ipv4 and ipv6 addresses.

    The next step would be to adapt ypcat and related utilities.

    The trickiest part would probably be to make NSS work with NIS/IPv6 - libnss_nis needs to be adapted to ti-rpc, and linked against it.

    Finally, other NIS tools such as yppasswd, yppasswdd, ypxfrd etc need to be made IPv6 aware.

    People

    Olaf Kirch originated this idea and has started to work on this.
