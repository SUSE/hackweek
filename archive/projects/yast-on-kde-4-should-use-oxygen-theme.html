    • YaST on KDE 4 should use Oxygen theme

    Tags: openSUSE, kde4, idea, theme, theming, oxygen

    Description

    Currently, YaST on KDE 4 is using a Plastique spin-off theme. This gives YaST the look of a QT3 application, and makes it look out-of-place on KDE 4's Oxygen desktop. I don't think there would be much work involved to switch the theme over to Oxygen. One programmer should easily be able to take on this small project.

    People

    curtis howard originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
