    • Find out build dependencies

    Tags: Idea, buildservice

    Description

    Finding out build dependencies is a problematic process, because you need to read log files and then try to add some devel packages and recompile. We need a tool to make this easier. In the past I tried LD_PRELOAD, but it's too fragile, so I'd suggest strace on the rpmbuild process and then looking around for all not found files in INDEX.gz

    People

    Stephan Kulow originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
