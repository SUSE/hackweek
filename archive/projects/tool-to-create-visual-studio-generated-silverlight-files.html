    • Tool to create Visual Studio generated Silverlight files

    Tags: Idea, devtools, mono

    Description

    When creating a silverlight project in Visual Studio, the IDE will generate some files to bind between the XAML and C# code. This allows you to do something like this:

    In XAML: ...

    and then in C#:

    my_rect.Height = "100"

    Essentially all the generated file does is create a partial class like this:

    Rectangle my_rect;

    private void InitializeComponent () { my_rect = FindName ("my_rect") as Rectangle; }

    People

    Jackson Harper originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
