    • Make tablets work out of the box

    Tags: Idea, openSUSE, SLED, X11, hardware, graphics

    Description

    When a person plugs a tablet (usually Wacom) into a computer running Linux, it should automatically do the right thing and work. It should not work as simply a pointer (in relative mode), but should function like a tablet. This means the top left should map to the screen's top left, the bottom right should also be on the bottom right of what is displayed. Pressure sensitivity should work. The device should be hot-pluggable and X should just automatically do the right thing. No extra configuration should be necessary.
