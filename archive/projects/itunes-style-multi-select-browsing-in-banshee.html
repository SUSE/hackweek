    • iTunes style multi-select browsing in Banshee

    Tags: Idea, InProgress, TryMe, JudgeBest, JudgeDesktop, JudgeMe, Desktop, SLED, Banshee, Mono

    Description

    In iTunes, users can select music based on multiple genres, artists, and albums. This view is especially useful when you have a large music library, and you know what genre(s)/artist(s)/album(s) you want to listen to but you don't want to create a new playlist. It would be nice if Banshee had the same or similar view.

    People

    Rusty Howell originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
