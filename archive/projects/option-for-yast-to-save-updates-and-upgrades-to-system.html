    • Option for YaST to save updates and upgrades to system

    Tags: Idea

    Description

    YaST upon running of an update or upgrade would allow the option to save the updates and upgrades so that a local repository could be made and saved. This then can be used to create a end user update/upgrade DVD or local repo on a other machine.

    People

    Dean Hilkewich originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
