===== Description =====
SMB Traffic analyzer is a project to audit a Samba server in a graphical way. It can be used to watch Samba traffic over a defined period of time. Find a detailed description of the possibilities and the technical details on it's homepage.

The "Idea" is to use Hackweek time to improve and develop SMB Traffic analyzer to get it out of it's alpha state and become a useable tool.

We'll regulary update it's homepage with news from the development progress.

Homepage is here: [[http://holger123.wordpress.com/smb-traffic-analyzer/|SMB Traffic analyzer homepage]]
